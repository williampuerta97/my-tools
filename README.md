# my-tools

Colección de herramientas web **estáticas** (HTML, CSS y JavaScript en el navegador). No requiere instalación ni servidor: abre los archivos con tu navegador o sirve la carpeta con cualquier servidor local si lo prefieres.

## Punto de entrada

- **[index.html](index.html)** — Página principal con enlaces a las herramientas.

## Herramientas

| Archivo | Descripción |
|--------|-------------|
| [simulador_credito_abonos.html](simulador_credito_abonos.html) | Simulador de crédito con tabla de amortización, abonos extraordinarios (reducción de cuota o de plazo), seguros y otros conceptos mensuales. Entradas en formato moneda COP. |
| [app_finanzas_personales_mensuales.html](app_finanzas_personales_mensuales.html) | Finanzas personales por mes: ingresos, gastos, ahorros y totales. Los datos se guardan en **localStorage** del navegador; incluye exportar/importar respaldo en JSON. |

## Uso rápido

1. Clona o descarga este repositorio.
2. Abre `index.html` (doble clic o “Abrir con” el navegador), o abre directamente cualquier `.html` de la lista anterior.

> **Nota:** Si abres los HTML como `file://`, todo suele funcionar igual; el almacenamiento local (`localStorage`) es por origen y por navegador.

## Requisitos

- Navegador moderno con JavaScript habilitado.
